Interpreting the signals:

  • Signal Quality: Excellent (≥-85 dBm) → Good (≥-95) → Fair (≥-105) → Weak
  • Registration (CREG): 0=Not registered, 1=Home, 2=Searching, 5=Roaming
  • Packet Domain (CEREG): Same scale. LTE registration separate from voice.
  • Data Attach (CGATT): 0=Detached, 1=Attached. Must be 1 for data.
  • Slow NETLIGHT blink (15s+): Usually indicates mode active but idle. Check power supply (should be ≥4V @ 800mA capacity).
RF Interference Test

Disable GPS to check if it's interfering with LTE signal. Watch RSSI and registration before/after. If signal improves when GPS is off, RF interference is likely.

ELQDrone is a field console built around ESP32, SIM7600G, GPS, SMS, and SD-hosted web tools. It was designed to support practical modem control, telemetry, and location-aware messaging.

Project focus

  • SD-hosted portal served over the device Wi-Fi AP
